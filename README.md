# Companies Directory ‚Äî React + Redux Toolkit Query + Node/Express

A fully responsive companies directory application built with **React**, **Vite**, **Redux Toolkit Query**, **TailwindCSS**, and an **Express.js backend**.  
This project demonstrates clean architecture, API integration, pagination, infinite scroll, filtering, sorting, and modern UI development best practices.

---

## üöÄ Features

### **Frontend (React + Vite + Redux Toolkit)**

- Responsive UI with **Grid + Table view**
- Search, filter (location, industry), and sort by name
- **Pagination (page mode)**
- **Infinite Scroll (lazy loading)**
- Global state management using **Redux Toolkit + RTK Query**
- Clean UI built with **TailwindCSS**
- Dedicated reusable components (FiltersBar, TableView, CardItem)
- Error & loading states

### **Backend (Node + Express)**

- REST API for fetching companies
- Server-side pagination
- Filtering, searching & sorting
- CORS enabled for frontend consumption
- Lightweight JSON-based mock database
